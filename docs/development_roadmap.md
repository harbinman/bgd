## 里程碑 3：硬件引擎与画中画系统 (Hardware Core)
- [ ] **3.1 相机流挂载 (PIP)**: 实现可拖拽、缩放的画中画预览窗，确保帧率不低于 30fps。
- [ ] **3.2 亮度控制桥接**: 编写 Android Native 代码，实现 App 运行期间的系统最高亮度锁定与 WakeLock。
- [ ] **3.3 背景图形引擎**: 实现 **Catchlight Layer**。初步支持“心形眼神光”模版及其缩放交互。

## 里程碑 4：功能十二宫格全逻辑实现 (12-Grid Full Features)
- [ ] **4.1 核心光影控制组**:
    - **全屏 (Full Screen)**: 一键入流，全屏纯色补光切换。
    - **色卡 (Color Card)**: 12 类摄影色温滑动选择与实时预览。
    - **亮度 (Brightness)**: 系统级亮度滑杆逻辑与垂直滑动手势绑定。
- [ ] **4.2 拍摄参数调节组**:
    - **比例 (Ratio)**: 9:16 / 4:3 / 1:1 动态架构调整。
    - **高清 (HD)**: 分辨率切换与存储质量控制。
    - **缩放 (Zoom)**: 数码变焦算法与 UI 指挥逻辑。
- [ ] **4.3 拍摄动作与声音组**:
    - **单拍 / 倒计时 (Delay)**: 3s/5s/10s 倒计时动画与快门触发。
    - **自拍 (Selfie)**: 前后摄像头切换与自适应补光策略。
    - **音效 (Sound)**: 快门声音包管理与静音切换。
- [ ] **4.4 扩展与个性化**:
    - **爱心眼 (Catchlights)**: 眼神光图形参数调节（粗细、透明度）。
    - **更多 (More)**: 集成辅助水平仪、网格线与镜像开关。
    - **语言切换 (i18n)**: 中英文状态机切换与持久化记录。
