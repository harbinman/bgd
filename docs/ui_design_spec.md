# 《喵喵补光灯》UI 设计系统规范 (v1.1)

本设计旨在打造一个**时尚、年轻、女性化**的视觉体系，将冷思考的硬件工具转化为感性、精致的生活方式应用。

## 1. 视觉主题：灵动极简 (Fluid Minimalist)
- **核心感官**: 既有高科技的严谨（深色背景），又有如化妆笔、香水瓶般的时尚感。
- **关键技术**: 采用 **Liquid Glass (流体玻璃)** 效果，增加界面的通透感和层次感。

## 2. 设计规范 (Design Tokens)

### 2.2 色彩体系 (Color Palette)
- **主体底色**: `Charcoal Gray (#121212)` - 确保补光效果不被中性灰干扰，且显得专业、高级。
- **女性化强调色**: `Pearl Pink (#FFD1DC)` - 用于图标高亮、进度条及核心心形补光形状。
- **辅助点缀色**: `Frosted Lavender (#E6E6FA)` - 用于玻璃质感的微弱反光层。

### 2.3 几何与形状 (Geometry)
- **极简圆角**: 所有容器和按钮采用 `24dp` 以上的高圆角矩形，营造柔和、安全、亲和感。
- **眼神光轮廓**: 预设的“爱心眼”采用非闭合的柔性路径绘制，边缘带有微弱的内外发光 (Inner/Outer Glow)，确反射在瞳孔中时显得自然而深邃。

### 2.4 字态 (Typography)
- **品牌/标题**: `Playfair Display` (Serif) - 赋予类似时尚杂志（Vogue/Harper's Bazaar）的排版质感。
- **功能文案**: `Outfit` 或 `Inter` (Sans-Serif) - 确保在不同屏幕亮度下的可读性。

## 3. 页面参考设计 (UI Mockups)

### 3.1 核心补光页 (Main Screen)
- **布局特点**: 沉浸式全黑背景。
- **视觉重心**: 中心为发光的粉色心形（眼神光模版）。
- **画中画**: 底部采用磨砂玻璃质感的取景框，四角带有精致的高光。

### 3.2 12 宫格工具栏 (Utility Grid)
- **触发机制 (Trigger)**: 
    - **视觉入口**: 位于**屏幕顶部的下拉胶囊**（`↓ 工具`），带毛玻璃背景与 0.5pt 极细边框。
    - **交互逻辑**: 点击触发 `_isMenuVisible` 状态切换，菜单以 **Alignment.topCenter** 锚定，并通过负 Y 轴偏移实现**从顶向底**的下滑弹入（Top-down Slide）。
    - **收起方式**: 
        - 点击菜单外阴影区域
        - 再次点击触发胶囊
        - **[NEW]** 向上滑动手势收起：在菜单容器上向上拖拽超过 120px 或快速向上滑动（速度 < -600px/s）即可关闭
        - **[NEW]** 拖拽手柄视觉反馈：拖拽时手柄宽度从 40px 扩展到 60px，透明度从 0.2 提升到 0.4
- **图标风格**: 采用极细的 `0.5pt` 线性图标（Outline Style），轻盈而不沉重。
- **容器效果**: 整体面板采用高斯模糊（Gaussian Blur 30px）效果，面板底部带有 40dp 大圆角，并应用统一颜色的粉色微光边框（0.08 Opacity），避开了 Flutter 渲染引擎对非统一边框圆角的限制。
- **布局优化 (Layout Optimization) [NEW]**:
    - **容器高度**: 从屏幕高度的 72% 增加到 78%，确保 12 个网格项完整显示
    - **内部间距**: 顶部 padding 从 60px 优化到 50px，网格行间距从 24px 优化到 20px
    - **网格宽高比**: 添加 `childAspectRatio: 0.85` 控制，确保网格项在不同屏幕尺寸下均能完整显示
    - **无滚动设计**: 保持 `NeverScrollableScrollPhysics()`，所有 12 个工具项直接可见，无需滚动

### 3.3 心形眼神光视觉标准 (Heart Catchlight Visual Standard) [NEW]
- **整体形象**: 心形应呈现为一个**大面积亮粉色发光气球**，而非仅有轮廓线。参考图中心形光源具有填满内部的柔和粉色实心效果。
- **尺寸**: 心形宽度约占屏幕宽度 **80%**，视觉上占据屏幕中心主导地位。
- **内部填充**: 采用**径向渐变 (RadialGradient)** 从中心向外扩散；中心为高亮粉白色，边缘渐变为深玫瑰红，最外层过渡到黑色背景。
- **外发光**: 心形外侧有大面积柔和光晕（`MaskFilter.blur` sigma >= 30），模拟真实补光灯的环境散射效果。
- **背景环境光**: 屏幕底部需有一个半圆形玫瑰红渐变光晕作为烘托，增强整体发光氛围感。
- **呼吸动效**: 保持现有 0.8x-1.2x 的脉冲动画，频率约 4 秒一个循环。

### 3.4 底部操作区布局规范 (Bottom Action Zone) [NEW]
底部区域从上至下分为三个层次：

**第一层：快捷工具行 (Shortcut Toolbar)**
- 位于 PIP 相机预览窗正下方，横向排列 4 个操作入口：
  - `Beauty`（美颜，左侧）— 线性图标 + 文字标签
  - 大圆形快门键（居中）— 粉色圆形背景，直径约 64dp，视觉最突出
  - `Filter`（滤镜）— 线性图标 + 文字标签
  - `Timer`（定时）— 线性图标 + 文字标签

**第二层：底部主导航栏 (Bottom Tab Bar)**
- 毛玻璃圆角胶囊容器，内嵌 4 个标签页：
  - `Home` — 首页
  - `Fill Light`（当前选中，粉色心形高亮）— 主拍摄页
  - `Filters` — 滤镜风格页
  - `Gallery` — 样片库
- 选中项下方有粉色指示条，容器背景为半透明黑色毛玻璃。
- PIP 取景框圆角规格：由 24dp 调整至 **12dp**（更接近专业相机取景框风格）；边框由实线发光改为**粉色虚线描边**。

### 3.5 品牌引导页 (Brand Intro)
- **视觉重心**: 中心放置一张大尺寸、高面值的“喵喵”3D 插画。
- **功能逻辑**: 此页面作为 App 的视觉门面，不再直接弹出授权。
- **按钮设计**: 放弃强迫式的授权文案，改为“立即开启”等具有期待感的按钮，并带有温润的粉色渐变效果。
